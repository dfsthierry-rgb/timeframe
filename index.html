📊 Importação de Excel (.xlsx)

Selecione um arquivo Excel (.xlsx) da sua planilha. O cabeçalho deve estar na primeira linha.

Formatos aceitos: .xlsx, .xls

O arquivo permanece no seu computador. Nenhum dado é enviado para servidores.

Requisito:

No Google Sheets: Arquivo → Publicar na Web (ou disponibilize para acesso público). Depois cole o link abaixo.

Aceita links do tipo: docs.google.com/spreadsheets/d/...

Se a planilha for restrita, o navegador pode bloquear o acesso. Nesse caso, use a opção Colar dados.
